## A Note about AI

This collection is a set of essais on prompt engineering as a craft. Each essai has its own *A note about AI* on the local question. The unifying principle: **prompting is the craft; the model is the substrate, not the practitioner.** Each essai treats a specific aspect of working with the model — the Chinese Room, persona patterns, sycophancy, ReAct — and each note names where the model can scaffold the discussion and where the substantive thinking has to be yours. The essais are a way of thinking carefully about a tool while using it. The thinking is what makes the use a craft rather than a habit.
